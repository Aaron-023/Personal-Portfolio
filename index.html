    Premier League Magic 8-ball

    The purpose of this Codecademy project was to be able to showcase my JavaScript skills by generating a random output for various sources of data.

    I chose the Premier League as my inspiration as it contains various teams within various positions of a table. My three outputs to return a random string were the team, the teams position within the table and finally a phrase that a friend might use in conversation regarding the teams placement in the league.

    You can view Premier League Magic 8-ball on Github
